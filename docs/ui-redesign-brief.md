# ChromaBlocks UI Redesign Brief

> Pencil 작업용 디자인 가이드. 각 화면별 상세 스펙 + 레퍼런스.

## 📐 기본 스펙

| 항목 | 값 |
|------|-----|
| 해상도 | **393 x 852 px** (세로 고정, 모바일) |
| 보드 | **8x8** 격자 |
| 트레이 | 피스 3개 (Expert Lv30+: 2개) |
| 색상 수 | 6색 (기본 7색이나 SPECIAL 제외) |
| 테마 | 다크 배경 기본 |

## 📱 화면 목록 (총 7개)

---

### 1. 🌅 스플래시 화면 (Splash)
**목적:** 앱 시작 시 브랜딩

**요소:**
- ChromaBlocks 로고 (중앙)
- 색상 블록 아이콘/마스코트
- 로딩 인디케이터 (선택)
- 0.5초 후 터치 스킵 가능

**디자인 방향:**
- 미니멀, 임팩트 있게
- 6색 블록이 모여서 로고를 형성하는 느낌
- 다크 배경에 블록 색상들이 빛나는 효과

---

### 2. 🏠 홈 화면 (Home)
**목적:** 메인 메뉴, 진입점

**버튼 (위→아래 순서):**
1. **▶ New Game** — 주 CTA, 가장 크고 눈에 띄게
2. **▶ Continue** — 저장된 게임 있을 때만 표시
3. **📅 Daily Challenge** — 날짜 표시 + 완료 뱃지
4. **❓ How to Play** — 튜토리얼

**하단:**
- 🔊 사운드 토글
- ⚙️ 설정

**통계 섹션:**
- Best Score: `000,000`
- Games Played: `000`
- Avg Score: `000,000`

**디자인 방향:**
- 깔끔한 카드 기반 레이아웃
- New Game 버튼은 그라데이션 (Sky→Lavender) + 글로우
- 통계는 하단에 작게, 미니 카드 3열
- Daily Challenge는 캘린더 아이콘 + 스트릭 표시

---

### 3. 🎮 인게임 HUD
**목적:** 게임 중 정보 표시

**상단 영역 (위→아래):**
- **현재 점수** — 크게, 볼드, 중앙 (카드 배경)
- **레벨 + 프로그레스 바** — 좌측
- **최고 점수** — 우측, 작게

**중앙:**
- **8x8 보드** — 게임의 핵심, 최대한 넓게
- 보드 좌우 패딩: 32px
- 위기 시 테두리 색상 변경 (70%→주황, 80%→빨강 맥동)

**하단:**
- **피스 트레이** — 3개 피스 가로 배치
- **⏸ 일시정지 버튼** — 좌상단 또는 우상단
- **콤보 표시** — 보드 위에 팝업 (COMBO x3 등)

**디자인 방향:**
- 점수 카드: 반투명 글래스모피즘
- 보드: 셀 사이 미세한 갭, 둥근 모서리
- 피스 트레이: 하단 고정, 약간 들어올린 카드
- 콤보: 네온 글로우 텍스트, 스케일 바운스

---

### 4. ⏸️ 일시정지 화면 (Pause)
**목적:** 게임 중단, 재개/종료 선택

**요소:**
- "PAUSED" 타이틀
- ▶ Resume (재개)
- 🏠 Home (홈으로)
- 현재 점수 / 레벨 표시

**디자인 방향:**
- 보드 위 반투명 오버레이 (배경 블러 또는 딤)
- 중앙 모달 카드
- 미니멀하게

---

### 5. 💀 게임오버 화면 (Game Over)
**목적:** 결과 확인, 재도전/광고 유도

**요소:**
- "GAME OVER" 타이틀
- **최종 점수** — 크게, 카운팅 애니메이션
- **최고 점수** — 신기록 시 🏆 뱃지
- **레벨 달성**
- **라인 클리어 수**

**버튼:**
1. 🔄 **Play Again** — 주 CTA
2. 📺 **Continue (Ad)** — 보상형 광고로 이어하기 (게임당 1회)
3. ✨ **Double Score (Ad)** — 점수 2배 (게임당 1회)
4. 🏠 **Home**

**디자인 방향:**
- 점수가 히어로 — 거대한 숫자
- 신기록 시 골드 글로우 + 파티클
- 광고 버튼은 구분되게 (아이콘 + "Watch Ad" 라벨)
- 통계는 아이콘 + 숫자 미니 카드

---

### 6. ⚙️ 설정 화면 (Settings)
**목적:** 사운드, 테마, 광고 제거

**항목:**
- 🔊 사운드 ON/OFF 토글
- 📳 진동 ON/OFF 토글
- 🎨 테마 선택 (5종: 기본/파스텔/네온/오션/선셋)
- 🚫 광고 제거 (IAP)
- ℹ️ 버전 정보

**디자인 방향:**
- 리스트 형태, 각 항목 카드
- 테마는 미리보기 색상 원형 스와치
- 토글은 iOS 스타일 스위치

---

### 7. 📖 튜토리얼 오버레이 (Tutorial)
**목적:** 첫 플레이 가이드

**5단계:**
1. 피스를 드래그해서 보드에 놓기
2. 가로/세로 줄 완성하면 클리어
3. 콤보로 높은 점수
4. 피스 교체 (스왑) 기능
5. 게임오버 조건

**디자인 방향:**
- 반투명 오버레이 + 스포트라이트 (해당 영역만 밝게)
- 하단에 설명 텍스트 + "다음" 버튼
- 손가락 드래그 애니메이션 (1단계)

---

## 🎯 전체 디자인 원칙

1. **다크 모드 우선** — 딥 네이비/차콜 배경
2. **글래스모피즘** — 반투명 카드, 미세한 블러
3. **네온 액센트** — 블록 6색을 UI 포인트로 활용
4. **라운드** — 모든 요소 둥근 모서리 (12~20px radius)
5. **여백** — 답답하지 않게, 충분한 패딩
6. **모바일 터치** — 최소 터치 영역 44x44px
7. **애니메이션 암시** — 글로우, 그라데이션으로 동적 느낌
