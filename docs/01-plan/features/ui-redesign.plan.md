# Plan: BlockDrop UI Redesign

> **Feature**: ui-redesign
> **Date**: 2026-02-08
> **Status**: Draft
> **Goal**: 현재 딱딱하고 밋밋한 UI를 세련되고 플레이 욕구를 자극하는 디자인으로 개선

---

## 1. 현황 분석 (As-Is)

### 1.1 현재 문제점

| 영역 | 문제 | 심각도 |
|------|------|--------|
| 블록 디자인 | 단순 그라디언트, 입체감/깊이감 부족 | 높음 |
| 배경 | 단색 (#1A1B2E) 정적 배경, 생동감 없음 | 높음 |
| 홈 화면 | 모드 카드가 단조로움, 시각적 자극 부족 | 높음 |
| 게임 HUD | 기능적이지만 시각적 매력 없음 | 중간 |
| 피드백 애니메이션 | 라인 클리어 플래시만 존재, 콤보/점수 팝업 없음 | 높음 |
| 게임 보드 | 그리드 라인만 보임, 보드 프레임/테두리 디자인 없음 | 중간 |
| 게임 오버 화면 | 정적, 감정적 임팩트 부족 | 중간 |

### 1.2 현재 컬러 팔레트

```
배경: #1A1B2E (너무 어둡고 단조)
카드: #252742 (배경과 대비 부족)
보드: #2D2F4E (구분감 약함)
Primary: #6C5CE7 (보라 단색)
```

## 2. 벤치마킹

### 2.1 Tetris Effect (최고 비주얼)

| 요소 | 특징 |
|------|------|
| 배경 | 동적 파티클 배경, 테마별 변화 (바다/우주/숲) |
| 블록 | 반투명 글로우 효과, 네온 라인 |
| 클리어 이펙트 | 줄 클리어 시 파티클 폭발 + 음악 싱크 |
| 철학 | "흥분과 이완의 사이클" → 몰입 (Flow State) |

### 2.2 Block Blast (다운로드 1위, 2025: 3.56억)

| 요소 | 특징 |
|------|------|
| 컬러 | 부드러운 순환 팔레트 (블루→그린→퍼플→오렌지) |
| 레이아웃 | 깔끔한 미니멀 그리드, 명확한 시각적 계층 |
| 피드백 | 미세 애니메이션 + 햅틱 + 사운드 (배치/클리어 시) |
| 철학 | "Less is More" — 심플하지만 만족스러운 경험 |

### 2.3 2025-2026 모바일 게임 UI 트렌드

1. **미니멀리즘 + 최대 임팩트**: 깔끔하되 시각적으로 강렬
2. **다이제틱 UI**: UI가 게임 세계에 자연스럽게 녹아듦
3. **제스처 기반**: 버튼 최소화, 직관적 터치 조작
4. **글로우/파티클**: OLED 화면에서 빛나는 네온+파티클 효과
5. **테마 전환**: 레벨/시간별 배경/컬러 전환으로 시각적 새로움

## 3. 리디자인 방향

### 3.1 디자인 컨셉: "Luminous Flow"

> **깨끗하면서도 빛나는, 심플하면서도 살아있는**

핵심 원칙:
- **Glow & Depth**: 블록과 UI 요소에 발광 효과와 깊이감
- **Living Background**: 정적 배경 → 은은하게 움직이는 파티클/그라디언트
- **Micro-Delight**: 모든 인터랙션에 미세한 시각적 보상
- **Theme Cycling**: 레벨/모드별 컬러 테마 전환

### 3.2 새로운 컬러 시스템

```
── 기본 팔레트 ──
배경:     #0D1117 → #161B22 (깊은 다크, GitHub Dark 참고)
서피스:   #21262D (카드/패널)
보드:     #0D1117 (배경과 동일, 테두리로 구분)
Primary:  #7C3AED → #A78BFA (보라 그라디언트)
Accent:   #06B6D4 (시안 글로우)

── 블록 컬러 (더 선명하고 글로우) ──
Coral:    #EF4444 → #FCA5A5 (글로우: #EF444440)
Amber:    #F59E0B → #FCD34D (글로우: #F59E0B40)
Lemon:    #EAB308 → #FDE047 (글로우: #EAB30840)
Mint:     #10B981 → #6EE7B7 (글로우: #10B98140)
Sky:      #3B82F6 → #93C5FD (글로우: #3B82F640)
Lavender: #8B5CF6 → #C4B5FD (글로우: #8B5CF640)
```

## 4. 화면별 리디자인 상세

### 4.1 홈 화면

**현재**: 단조로운 카드 4개 + 로고 + 하단 네비게이션
**목표**: 플레이 버튼이 중심, 시각적으로 매력적인 진입점

| 요소 | 변경 사항 |
|------|----------|
| 배경 | 정적 → 부유하는 반투명 블록 파티클 + 그라디언트 |
| 로고 | 텍스트 → 글로우 효과 + 미세 애니메이션 (펄스) |
| 플레이 버튼 | 크고 눈에 띄는 CTA, 글로우 링 효과 |
| 모드 선택 | 가로 스크롤 카드 (각 모드별 프리뷰 이미지) |
| Best Score | 하단 바 → 상단 배지 스타일 |
| Daily Challenge | 별도 배너 → 상단 팝 배너 + 카운트다운 |

### 4.2 게임 화면

**현재**: 기본 그리드 + 단색 블록 + 플래시 이펙트
**목표**: 몰입감 있는 플레이필드, 살아있는 비주얼

| 요소 | 변경 사항 |
|------|----------|
| 보드 프레임 | 그리드 라인만 → 글로우 테두리 + 미세 코너 장식 |
| 블록 | 플랫 → 내부 하이라이트 + 외부 글로우 + 미세 그림자 |
| 고스트 피스 | 반투명 → 점선 외곽선 + 미세 펄스 |
| 라인 클리어 | 화이트 플래시 → 글로우 웨이브 + 파티클 산란 |
| 콤보 | 텍스트만 → 스케일 팝업 + 컬러 버스트 |
| 점수 | 정적 → 증가 시 숫자 카운트업 애니메이션 |
| 배경 | 단색 → 레벨별 미세 그라디언트 변화 |
| HUD | 단순 텍스트 → 프로그레스 바 (레벨), 글로우 텍스트 |

### 4.3 게임 오버 화면

**현재**: 어두운 카드 + 정적 스탯
**목표**: 감정적 임팩트, 재시도 유도

| 요소 | 변경 사항 |
|------|----------|
| 진입 | 즉시 → 보드 붕괴 애니메이션 → 페이드 인 |
| 점수 | 정적 → 드럼 롤 카운트업 |
| 신기록 | 없음 → "NEW BEST!" 글로우 배지 + 파티클 |
| CTA | 두 버튼 → 큰 RETRY 버튼 (글로우) + 작은 홈 |
| 감정 | 밋밋함 → 아쉬움(점수 표시) + 도전 의욕(재시도) |

## 5. 구현 범위 (Phase 1 - Pencil 디자인)

Pencil에서 먼저 3개 화면을 리디자인하고, 코드 구현은 Design 문서 이후.

### Phase 1: Pencil 디자인 리뉴얼

| # | 항목 | 설명 |
|---|------|------|
| 1 | 홈 화면 리디자인 | 새 컬러, 글로우 효과, 플레이 중심 레이아웃 |
| 2 | 게임 화면 리디자인 | 새 블록 스타일, 보드 프레임, HUD 개선 |
| 3 | 게임 오버 리디자인 | 임팩트 있는 스탯 + 재시도 유도 |

### Phase 2: 코드 구현 (Design 문서 이후)

| # | 항목 | 설명 |
|---|------|------|
| 1 | 새 컬러 시스템 | AppColors 리팩토링 |
| 2 | 블록 렌더링 개선 | 글로우 + 하이라이트 + 그림자 |
| 3 | 애니메이션 시스템 | 라인 클리어 파티클, 콤보 팝업, 점수 카운트업 |
| 4 | 배경 시스템 | 레벨별 그라디언트 배경 전환 |
| 5 | 홈 화면 UI 개편 | 새 레이아웃 + 효과 |
| 6 | 게임 오버 개편 | 카운트업 + 신기록 배지 |
