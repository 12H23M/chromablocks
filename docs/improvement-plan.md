- **수정:** `splash_screen.tscn`
- **처방:**
  - 배경: radial gradient 중심 `#1E1245` → 외곽 `#0A0A1E`
  - "CHROMA" 왼쪽에서 slide_in (0.3s, cubic_out)
  - "BLOCKS" 오른쪽에서 slide_in (0.15s delay)
  - 3x3 미니 블록 그리드 pop_in (scale 0→1.1→1.0, stagger 0.05s)

### P0-2. 블록 렌더링 — 버블 → 플랫 블록
- **문제:** 흰색 원형 하이라이트가 "유리구슬" 느낌. Block Blast는 상단 그라디언트 밴드 사용
- **수정:** `DrawUtils.gd` → `draw_bubble_block()`
- **처방:**
  - 버블 하이라이트(흰색 원) 완전 제거
  - 상단 하이라이트: 블록 위쪽 30%에 밝은 그라디언트 (`color.lightened(0.15)`, alpha 0.6)
  - 하단 그림자: 아래 2px 어둡게 (`color.darkened(0.1)`, alpha 0.4)
  - 1px 내부 보더: `#FFFFFF` alpha 0.08
  - corner_radius: block_size × 0.18
- **컬러 팔레트 업데이트:**
  ```
  Pink:   #FF6B9D    Orange: #FF8C42    Green:  #6BCB77
  Blue:   #4D96FF    Yellow: #FFD93D    Purple: #9B72CF
  ```
  - HSL 통일: S 75-85%, L 55-65%

## 🎮 게임플레이 개선 (별도 트랙)

### Week 1: 피스 큐 + 홀드 시스템

**핵심 문제:** 평균 2,032 vs 최고 19,395 = 운 의존도 높음. 전략적 선택권 부족.

**설계:**
```
트레이:  [피스A] [피스B] [피스C]
다음:    [피스D] [피스E] [피스F]  ← 50% 투명도 실루엣
홀드:    [____]  ← 1칸, 트레이당 1회 교체
```
- 기존 Piece Swap 제거 → 홀드로 완전 대체
- 다음 트레이 미리보기: 현재 3개 배치 완료 시 나올 다음 3개 실루엣 표시
- **목표:** 평균 점수 2,032 → 3,500~4,000

### Week 2: 클리어 피드백 재설계

**현재:** 피스 배치 → 전부 즉시 발생 (0.3s) → 뭐가 뭔지 모르게 끝남

**개선 타이밍:**
```
피스 배치
  → 0.15s 대기 (anticipation)
  → 라인 클리어 (0.3s, 빛나며 사라짐)
  → 0.2s 대기
  → Chroma Chain (0.4s, 순차 제거)
  → 0.2s 대기
  → Chroma Blast (0.5s, 전체 폭발)
  → 점수 카운터 롤업 (0.3s)
전체 약 1.5~2.0초 — "와, 연쇄 터진다!" 체감
```

**점수 카스케이드:**
```
라인 클리어:    +120  (흰색, 작은 폰트)
Chroma Chain:  +340  (색상별, 중간 폰트)
Chroma Blast:  +890  (색상별, 큰 폰트, 화면 흔들림)
TOTAL:         +1,350 (금색, 0.5s 후 통합)
```

**Perfect Clear 추가:** 보드 0개 블록 → ×10,000점 + 레인보우 이펙트

### Week 3: 미션 런 모드

**설계:**
```
게임 시작 시 선택:
  [자유 모드]  ← 현재와 동일
  [미션 런]   ← 한 게임에 미션 3개 동시 활성화
```

미션 예시:
- "3줄을 클리어하세요" → 50 XP
- "콤보 x3을 달성하세요" → 80 XP
- "Chroma Chain을 발동하세요" → 120 XP

미션 난이도 스케일링:
- 0~1,000점: 기본 미션 풀 (쉬움)
- 1,000~5,000점: 중급 미션 풀
- 5,000점+: 고급 미션 풀

게임오버 시: "미션 7개 완료! 최고: 12개" → 별도 성취축

### Week 4: 밸런싱 + 폴리시

**제거/단순화 목록:**
| 항목 | 조치 | 이유 |
|------|------|------|
| Piece Swap | 제거 → 홀드 대체 | 상위호환, 공존하면 혼란 |
| DDA Mercy | 약화 (75%→85%) | 너무 관대하면 성취감 저하 |
| 성능 등급 B/C | 제거 → S/A만 | B/C는 기분만 나쁨 |
| 보드 프리필 | 선택적 | 자유모드=빈보드 옵션, 프리필=데일리 전용 |
