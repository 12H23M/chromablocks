## 분석

### 🏆 추천: Spread 40% + 2 near-complete

**이유:**
1. **최고 점수** (27,300 avg) — 현재 대비 +341점 (+1.3%)
2. **빠른 첫 클리어** (1.9턴 avg) — 현재 2.7턴보다 30% 빠름
3. **높은 3턴이내 클리어율** (81.7%) — 현재 61.8% 대비 +20pp
4. 라인클리어 수도 가장 많음 (146.1)

### 심리적 임팩트

- 현재 전략: 첫 클리어까지 평균 2.7턴 → 약간 기다림
- Spread 40%: 첫 클리어까지 평균 1.9턴 → **거의 즉시 피드백**
- 3턴이내 클리어율 +20pp는 **첫 10초의 임팩트** 크게 개선

### 콤보 분석

5+콤보 도달율은 현재 전략(54.2%)이 약간 높지만 모든 전략이 50%대로 유사.
중요한 것은 콤보 경험 자체는 동일하게 보장됨.

### 리텐션 주의

모든 전략에서 AI가 300턴 한도에 도달함 (게임오버 없음).
실제 플레이어는 AI보다 훨씬 일찍 게임오버 → 보드 밀도 대비 난이도 조절 여지 있음.

## 권고 사항

1. **즉시 적용:** Spread 40% + 2 near-complete 전략으로 prefill 교체
   - `piece_generator.gd`의 `prefill_board()` 로직 수정
   - 하단 집중 → 전체 분산 + 2개 near-complete 라인 확보

2. **A/B 테스트 가능:** 현재 전략 대비 리텐션 지표 추적

3. **콤보 보너스 검토:** 5+콤보 도달율 50%대는 적정 수준
