## 시장 개요

블록 퍼즐 장르는 2024-2025년 모바일 게임 시장에서 가장 빠르게 성장한 캐주얼 장르 중 하나. Block Blast가 2024년 US App Store 무료 게임 1위를 장기간 유지하며 장르 전체의 주목도를 끌어올림. 핵심 트렌드:

- **다크 모드 우세**: 상위 게임 70% 이상이 다크/딥 배경 사용
- **미니멀리즘**: 불필요한 UI 요소 최소화, 보드에 집중
- **촉감 피드백**: 햅틱 + 시각적 만족감 (파티클, 스크린 셰이크)
- **원핸드 플레이**: 하단 트레이 + 드래그 앤 드롭 표준화
- **10x10 그리드**: Block Blast 이후 8x8→10x10 그리드가 시장 표준으로

### 1. Block Blast! (Hungry Studio)
> 2024-2025 최고 히트작. US 차트 1위 장기 유지.

#### 1-1. 컬러 팔레트
| 요소 | 색상 | 코드 (추정) |
|------|------|-------------|
| 배경 | 딥 네이비 블루 | `#1A1A2E` ~ `#16213E` |
| 보드 배경 | 약간 밝은 네이비 | `#1F2744` |
| 빈 셀 | 다크 그레이-블루 | `#2A2D4A` |
| 블록 - 빨강 | 비비드 레드 | `#E74C3C` |
| 블록 - 파랑 | 브라이트 블루 | `#3498DB` |
| 블록 - 초록 | 에메랄드 | `#2ECC71` |
| 블록 - 노랑 | 골드 옐로우 | `#F1C40F` |
| 블록 - 보라 | 퍼플 | `#9B59B6` |
| 블록 - 주황 | 오렌지 | `#E67E22` |
| 블록 - 핑크 | 핫핑크 | `#E84393` |
| 강조색 | 골드/화이트 | `#FFD700` / `#FFFFFF` |
| **전체 톤** | **다크 (딥 네이비)** | |

#### 1-2. 블록 스타일
- **타입**: 세미-플랫 + 미세한 그래디언트
- **그림자**: 블록 하단에 얇은 내부 그림자 (1-2px, 약간 어두운 동일 계열 색)
- **하이라이트**: 블록 상단-좌측에 미세한 밝은 하이라이트 (광택감)
- **라운딩**: `border-radius: 4-6px` (약간 둥근 사각형)
- **크기**: 셀당 약 32-36px (10x10 기준)
- **특징**: 블록 표면에 미세한 텍스처/노이즈 없음. 깔끔한 솔리드 컬러

#### 1-3. 보드 디자인
- **배경**: 보드 영역 자체가 약간 밝은 네이비로 구분
- **셀**: 빈 셀은 약간 밝은 사각형으로 표시 (미세한 인셋 느낌)
- **격자선**: 명시적 격자선 없음. 셀 간 약 1-2px 간격이 자연스러운 구분선 역할
- **프레임**: 보드 주변에 별도 프레임 없음. 배경색 차이로 영역 구분
- **그리드**: 8x8 (경쟁작 대비 약간 작은 그리드, 빠른 게임)

#### 1-4. HUD 레이아웃
- **점수**: 화면 상단 중앙, 큰 볼드 폰트 (32-40px)
- **최고점수**: 점수 아래 또는 우측에 작게 표시
- **설정/메뉴**: 좌측 상단 햄버거 또는 기어 아이콘
- **타이포그래피**: 산세리프, 굵은 웨이트 (Poppins/Nunito 계열), 흰색
- **여백**: 넉넉한 상단 여백 (노치/다이나믹 아일랜드 대응)

#### 1-5. 트레이/피스 영역
- **위치**: 화면 하단 1/4 영역
- **배치**: 가로 3열로 피스 나열
- **크기비율**: 보드 셀의 약 70-80% 크기로 축소 표시
- **배경**: 메인 배경과 동일 (별도 구분 없음, 미니멀)
- **인터랙션**: 드래그 시 피스가 원래 크기로 확대

#### 1-6. 이펙트
- **라인 클리어**: 클리어되는 줄이 하이라이트 → 파티클 폭발 (빛나는 조각들)
- **콤보**: 연속 클리어 시 화면 전체 번쩍임 + 콤보 텍스트
- **스크린 셰이크**: 대형 클리어 시 미세한 진동
- **드래그 피드백**: 블록이 보드 위에 올라가면 목적지 셀 하이라이트
- **배치 불가**: 붉은 기 또는 흔들림으로 불가 표시
- **게임오버**: 보드가 서서히 어두워지는 연출

#### 1-7. 전체 분위기
- **타겟**: 15-55세 초광범위 캐주얼 (특히 20-40대 여성)
- **감정**: 편안함 + 적당한 긴장감. 중독성 있는 "한 판 더" 루프
- **사운드**: 경쾌한 팝 사운드, 클리어 시 만족스러운 효과음
- **BGM**: 잔잔하거나 없음 (무음 플레이 최적화)

#### 1-8. 차별화 포인트
- **극단적 미니멀리즘**: UI가 거의 없음. 보드와 피스만 존재
- **다크 배경 + 비비드 블록**: 블록 가시성 극대화
- **빠른 세션**: 5분 내외 완결, 부담 없는 진입
- **광고 통합**: 자연스러운 리워드 광고 (부활, 힌트)
- **8x8 그리드**: 빠른 클리어 사이클로 도파민 루프 단축

### 2. 1010! Block Puzzle (Gram Games)
> 블록 퍼즐 장르의 원조격. 2014년 출시, 꾸준한 인기.

#### 2-1. 컬러 팔레트
| 요소 | 색상 | 코드 (추정) |
|------|------|-------------|
| 배경 | 따뜻한 크림/베이지 | `#FAF0E6` ~ `#F5E6D0` |
| 보드 배경 | 밝은 우드톤 | `#E8D5B5` |
| 빈 셀 | 연한 베이지 | `#E0D0B8` |
| 블록 - 파랑 | 소프트 블루 | `#5B9BD5` |
| 블록 - 빨강 | 코럴 레드 | `#E06666` |
| 블록 - 초록 | 세이지 그린 | `#7AB87A` |
| 블록 - 노랑 | 머스타드 | `#DBA84D` |
| 블록 - 보라 | 라벤더 | `#8E7CC3` |
| 강조색 | 다크 브라운 | `#5D4037` |
| **전체 톤** | **라이트 (따뜻한 뉴트럴)** | |

#### 2-2. 블록 스타일
- **타입**: 플랫 디자인 (초기) → 미세한 그래디언트 추가 (업데이트)
- **그림자**: 미세한 드롭 쉐도우
- **하이라이트**: 없거나 아주 미미
- **라운딩**: `border-radius: 2-4px` (거의 직각)
- **특징**: 단색 블록, 매우 깔끔. 초기 미니멀 디자인의 교과서

#### 2-3. 보드 디자인
- **배경**: 10x10 그리드, 따뜻한 베이지 톤
- **셀**: 얇은 라인으로 구분된 정사각형
- **격자선**: 연한 갈색/베이지 (1px)
- **프레임**: 얇은 외곽 라인 또는 미세한 그림자

#### 2-4. HUD 레이아웃
- **점수**: 상단 중앙, 큰 숫자 (다크 브라운)
- **최고점수**: 점수 옆 작은 텍스트
- **설정**: 좌측 상단
- **타이포그래피**: 깔끔한 산세리프, 미디엄 웨이트

#### 2-5. 트레이/피스 영역
- **위치**: 하단 1/3
- **배치**: 가로 3열
- **배경**: 메인 배경과 동일
- **크기**: 보드 셀의 약 60-70%

#### 2-6. 이펙트
- **클리어**: 줄이 페이드아웃 + 미세한 파티클
- **콤보**: 숫자 애니메이션 + 텍스트
- **전반적**: Block Blast 대비 절제된 이펙트

#### 2-7. 전체 분위기
- **타겟**: 25-60세 캐주얼 유저, 특히 여성
- **감정**: 차분함, 명상적, 편안한 시간 보내기
- **키워드**: Zen, 힐링, 단순함

#### 2-8. 차별화 포인트
- **따뜻한 톤의 선구자**: 우드/베이지 톤을 블록 퍼즐에 도입
- **10x10 그리드 표준 확립**: 이후 수많은 클론의 기준점
- **미니멀의 정석**: 불필요한 요소 제로
- **장수 비결**: 심플함 자체가 USP

### 3. Woodoku (Tripledot Studios)
> 우드 텍스처 + 스도쿠식 보드 = 프리미엄 느낌.

#### 3-1. 컬러 팔레트
| 요소 | 색상 | 코드 (추정) |
|------|------|-------------|
| 배경 | 다크 브라운/월넛 | `#2C1810` ~ `#3E2723` |
| 보드 배경 | 미디엄 우드톤 | `#5D4037` |
| 빈 셀 | 밝은 우드 | `#8D6E63` ~ `#A1887F` |
| 블록 색상 | 단일 컬러 (나무톤) | `#D4A76A` ~ `#C8956D` |
| 3x3 구분선 | 어두운 라인 | `#1A0E08` |
| 강조색 | 골드/앰버 | `#FFB300` |
| 스트릭 표시 | 파이어 오렌지 | `#FF6F00` |
| **전체 톤** | **다크 (프리미엄 우드)** | |

#### 3-2. 블록 스타일
- **타입**: 스큐모픽 (우드 텍스처)
- **그림자**: 블록에 입체적 드롭쉐도우 (3-4px)
- **하이라이트**: 나무 결 방향으로 자연광 반사 효과
- **라운딩**: `border-radius: 4-6px`
- **텍스처**: 실제 나무 결 텍스처 적용 (비트맵)
- **특징**: 같은 나무색이라 '색상'이 아닌 '형태'로 플레이

#### 3-3. 보드 디자인
- **배경**: 9x9 그리드 (3x3 박스 9개 = 스도쿠 레이아웃)
- **셀**: 둥근 인셋 스타일, 나무에 파인 홈 느낌
- **3x3 구분**: 두꺼운 어두운 라인으로 9개 섹션 구분
- **프레임**: 조각된 나무 프레임 느낌

#### 3-4. HUD 레이아웃
- **점수**: 상단 중앙, 골드 컬러, 큰 세리프 폰트
- **스트릭**: 점수 아래 "🔥 x5" 형태
- **설정**: 우측 상단 기어 아이콘
- **타이포그래피**: 세리프 또는 세미세리프 (프리미엄 느낌)

#### 3-5. 트레이/피스 영역
- **위치**: 하단 1/4
- **배치**: 가로 3열
- **배경**: 다크 우드 텍스처
- **크기**: 보드 셀의 약 70%

#### 3-6. 이펙트
- **줄 클리어**: 나무 조각이 부서지는 듯한 파티클 (나무 가루 느낌)
- **3x3 박스 클리어**: 특별한 이펙트 (빛나는 골드 파티클)
- **콤보**: 화염 이펙트 + 스트릭 카운터 증가
- **배치음**: 나무 블록이 놓이는 "톡" 소리

#### 3-7. 전체 분위기
- **타겟**: 30-60세, 프리미엄 캐주얼을 선호하는 유저
- **감정**: 고급스러움, 차분함, 공예적 만족감
- **키워드**: 프리미엄, 크래프트, 젠

#### 3-8. 차별화 포인트
- **스도쿠 + 블록**: 3x3 박스 클리어 보너스로 전략 깊이 추가
- **우드 텍스처**: 다른 게임과 확실한 시각적 차별화
- **프리미엄 포지셔닝**: 광고 최소화, 구독 모델 강조
- **스트릭 시스템**: 연속 플레이 동기부여

### 4. Blockudoku (Easybrain)
> 스도쿠 규칙 + 블록 퍼즐. Woodoku와 직접 경쟁.

#### 4-1. 컬러 팔레트
| 요소 | 색상 | 코드 (추정) |
|------|------|-------------|
| 배경 | 화이트/라이트 그레이 | `#F5F5F5` ~ `#FFFFFF` |
| 보드 배경 | 퓨어 화이트 | `#FFFFFF` |
| 빈 셀 | 연한 그레이 | `#E8E8E8` |
| 블록 - 기본 | 미디엄 블루 | `#4A90D9` |
| 3x3 구분 | 미디엄 그레이 | `#BDBDBD` |
| 강조색 | 브라이트 그린 | `#4CAF50` |
| 콤보 표시 | 골드/오렌지 | `#FF9800` |
| 다크모드 배경 | 다크 차콜 | `#1E1E1E` |
| **전체 톤** | **라이트 (클린 화이트) + 다크모드 지원** | |

#### 4-2. 블록 스타일
- **타입**: 플랫 + 미세 라운딩
- **그림자**: 최소한의 드롭쉐도우
- **하이라이트**: 없음
- **라운딩**: `border-radius: 4px`
- **특징**: 단일 컬러 블록 (블루 메인), 배치 후 색상이 약간 변경

#### 4-3. 보드 디자인
- **배경**: 9x9 그리드 (Woodoku와 동일 구조)
- **셀**: 깔끔한 사각형, 얇은 보더
- **3x3 구분**: 두꺼운 그레이 라인
- **프레임**: 미세한 그림자로 보드 영역 구분

#### 4-4. HUD 레이아웃
- **점수**: 상단 중앙
- **레벨/경험치**: 점수 아래 프로그레스 바
- **설정**: 좌측 상단
- **타이포그래피**: 깔끔한 시스템 산세리프

#### 4-5. 트레이/피스 영역
- **위치**: 하단 1/4
- **배치**: 가로 3열
- **배경**: 메인 배경과 동일
- **특징**: 피스 아래 미세한 그림자로 "들어올릴 수 있음" 암시

#### 4-6. 이펙트
- **클리어**: 줄이 펄스하며 사라짐 + 파티클
- **3x3 클리어**: 박스 전체가 빛남
- **콤보**: 텍스트 팝업 + 점수 애니메이션
- **전반적**: Woodoku보다 가볍고 깔끔한 이펙트

#### 4-7. 전체 분위기
- **타겟**: 25-50세, 퍼즐 좋아하는 캐주얼 유저
- **감정**: 깔끔함, 논리적, 약간의 도전감
- **키워드**: 클린, 로직, 캐주얼

#### 4-8. 차별화 포인트
