## 1. 경쟁작별 중독성 해부

### 1-1. Block Blast! (Hungry Studio)

**핵심 수치**: 일일 활성 유저 4,000만+, 월간 1억 2,000만+ (2024년 기준)
— *"Block Blast is arguably one of the most compelling casual games for mobile, as you can pretty much grab it anytime, anywhere. It has no ads, and it has virtually no 'premium' aspect."* (Pocket Gamer)

#### 중독 메커니즘 분석

**① 극단적 심플함 = 제로 진입 장벽**
- 규칙 설명 3초: "블록 배치 → 줄 완성 → 줄 사라짐"
- 튜토리얼 거의 없음. 첫 판이 곧 튜토리얼
- *"The concept is simple — Block Blast sounds ridiculously simple on paper, but in reality, it is a lot more complex."* (Pocket Gamer)

**② "체스 같은" 전략적 깊이**
- 항상 3~5수 앞을 내다봐야 함
- *"You always need to think 5 moves ahead (at least), and take into account every single possible block that could come."* (Pocket Gamer)
- **위험과 보상의 트레이드오프**: 큰 콤보를 노리면 보드가 위험해지고, 안전하게 가면 점수가 낮음
- 이 긴장감이 "한 판 더"를 만드는 핵심

**③ 게임오버 = "내 실수" 인식**
- 랜덤 요소가 있지만, 항상 "내가 더 잘했으면 됐는데"라는 감정 유발
- 게임오버 순간: 분노가 아니라 **아쉬움** → 즉시 재시작
- 점수가 숫자로 명확하게 보이기 때문에 "다음엔 이 점수를 넘기겠다"는 목표가 자동 생성

**④ 개인 최고기록(Personal Best) 추격**
- 리더보드 없이도 자체적인 "PB 갱신" 동기
- *"it's always worth trying again until you reach your personal best, isn't it?"* (Pocket Gamer)
- 유저들이 Reddit에 16만, 120만 점수 스크린샷을 공유하며 경쟁

**⑤ Adventure Mode (스토리 모드)**
- 레벨 기반 진행 + 내러티브 요소
- *"I'd wager that adventure mode is a major source of that success. A bit of a peek behind the curtain, but you'd be surprised how many developers credit adding stories or other narrative elements with helping them succeed."* (Pocket Gamer)
- June's Journey (Wooga)처럼 "드라마 같은 스토리라인"이 리텐션을 만듦

**⑥ 무광고 기본 경험**
- 강제 광고 없음 → 유저 신뢰 확보 → 자발적 보상형 광고 시청
- 이것이 4,000만 DAU의 비밀: **마찰 제거**

**⑦ 8×8 그리드의 빠른 리듬**
- 10×10 대비 보드가 작아서 줄 클리어 빈도 높음
- "도파민 히트" 간격이 짧음 → 만족감의 밀도가 높음
- 한 판 3~5분 → "짧아서 한 판만 더" 심리

### 1-2. 1010! (Gram Games)

**핵심**: 블록 퍼즐 장르의 원조 (2014). 수억 다운로드의 검증된 공식.

#### 중독 메커니즘 분석

**① 명상적 리듬 (Zen Flow)**
- 따뜻한 크림/베이지 색감 = 편안함
- 시간 제한 없음, 압박 없음
- "쉬면서 하는 퍼즐"이라는 포지셔닝 → 수면 전, 통근 중, 대기 중 플레이
- 유저 리뷰 공통 키워드: "relaxing", "zen", "calming"

**② 패턴 인식의 쾌감**
- 반복 플레이로 뇌가 "최적 배치 패턴"을 학습
- 초반: 의식적 사고 → 숙련 후: 무의식적 패턴 매칭 → **플로우 상태**
- 이 전환점을 경험하면 중독됨: "나 이거 잘하게 됐어"

**③ 10×10의 "완전 제어" 환각**
- 테트리스처럼 시간 압박이 없어서 "내가 완벽하게 통제하고 있다"는 느낌
- 실제로는 랜덤 블록 생성이지만, 느린 페이스가 "통제감"을 줌
- 통제감 = 스트레스 해소 → 일상에서 통제 불가한 것에 지친 사람들이 끌림

**④ 미니멀리즘 = 인지 부하 제로**
- UI 요소가 극소: 보드, 3개 블록, 점수
- "뭘 해야 하지?" 고민 0초 → 즉시 플레이 가능
- 앱 열고 3초 안에 게임 시작 가능

**⑤ 부정적 감정 최소화**
- 게임오버가 "실패"보다는 "이번 판은 여기까지"로 느껴짐
- 차분한 종료 → 부드러운 재시작 → 이탈 동기 약함

### 1-3. Woodoku (Tripledot Studios)

**핵심**: 블록 퍼즐 + 스도쿠 구조 = 전략적 깊이의 극대화.

#### 중독 메커니즘 분석

**① 이중 클리어 시스템 (줄 + 3×3 박스)**
- 9×9 보드를 3×3 구역 9개로 분할 (스도쿠 레이아웃)
- 줄 클리어 + 박스 클리어 = 두 가지 목표 동시 추구
- **선택의 풍부함**: "이 블록을 줄 완성에 쓸까, 박스 완성에 쓸까?"
- 이 선택이 모든 수에 전략적 깊이를 부여

**② 스트릭(Streak) 시스템 — 리텐션의 핵심**
- 매일 플레이하면 연속일수(🔥) 증가
- 스트릭이 깨지면 처음부터 → **손실 회피(Loss Aversion)** 활용
- "오늘 안 하면 30일 스트릭이 깨져" → 습관화의 강력한 도구
- 유저 리뷰: "I only play to keep my streak alive" — 이게 바로 성공

**③ 프리미엄 텍스처의 촉감적 만족**
- 우드 텍스처 = 스큐모픽 디자인의 따뜻함
- 블록 배치 시 "톡" 소리 = ASMR 효과
- "나무 조각을 맞추는" 물리적 쾌감을 디지털로 재현
- 유저 리뷰: "feels like putting real wooden pieces together"

**④ 3×3 박스 "완성"의 유혹**
- 박스가 거의 차면 → **자이가르닉 효과(Zeigarnik Effect)**
- 미완성 과제는 뇌가 계속 떠올림 → "이것만 채우면..." → 30분 지남
- Pocket Gamer 아티클의 인사이트: 스도쿠 식 "섹션 완성"이 추가 보상 레이어를 만듦

**⑤ 보너스 포인트의 변동 비율 강화**
- 줄 클리어: 기본 점수
- 박스 클리어: 보너스 점수
- 줄 + 박스 동시 클리어: 대형 보너스
- 연속 클리어 (콤보): 스트릭 보너스
- → 보상의 예측 불가능성 = **슬롯머신 효과**

## 3. 퍼즐 게임 중독성의 심리학

### 3-1. 핵심 심리 메커니즘

#### ① 변동 비율 강화 (Variable Ratio Reinforcement)
- **무엇**: 보상이 불규칙하게 주어질 때 가장 강한 행동 지속
- **게임 적용**: 1줄 클리어(기본) → 가끔 2줄 동시(보너스) → 아주 가끔 3-4줄 체인(잭팟)
- **Block Blast 사례**: 잘 깔아놓으면 한 번에 3-4줄이 연쇄적으로 사라지는 "빅 플레이" 발생 → 뇌가 이 순간을 추구하게 됨
- **ChromaBlocks 상태**: 콤보 시스템은 있지만, "빅 플레이"의 시각적/청각적 임팩트가 경쟁작 수준인가?

#### ② 자이가르닉 효과 (Zeigarnik Effect)
- **무엇**: 미완성 과제를 완성된 것보다 2배 더 잘 기억함
- **게임 적용**: "거의 다 채운" 줄이나 박스 → 뇌가 앱을 닫은 후에도 계속 생각
- **Woodoku 사례**: 3×3 박스가 8/9 채워져 있으면 앱을 끄기가 불가능
- **ChromaBlocks 적용 가능**: 현재 10×10 단일 보드에서는 "거의 완성" 상태가 줄 단위로만 존재 → 추가 완성 단위가 있으면 자이가르닉 강화

#### ③ 손실 회피 (Loss Aversion)
- **무엇**: 얻는 것보다 잃는 것을 2배 더 크게 느낌
- **게임 적용**: "지금 그만두면 이 보드 상태를 잃는다" / "스트릭이 깨진다"
- **Woodoku 사례**: 스트릭 시스템이 이것을 완벽히 활용
- **Block Blast 사례**: "이 높은 점수를 갱신할 기회를 놓친다"
- **ChromaBlocks 상태**: 손실 회피를 활용하는 시스템이 없음

#### ④ 자기 결정 이론 (Self-Determination Theory)
- **유능감(Competence)**: 스킬이 향상되는 느낌 → 점수 증가, 콤보 증가
- **자율성(Autonomy)**: 내가 선택한다는 느낌 → 블록 어디에 놓을지 결정
- **관계성(Relatedness)**: 다른 사람과 연결 → 리더보드, 소셜 공유
- **ChromaBlocks**: 유능감(△), 자율성(○), 관계성(✕)

#### ⑤ 유동적 난이도의 마법
- **Deconstructor of Fun 인사이트**: *"The question isn't 'how do we monetize our puzzle better?' The question is 'what kind of puzzle failure creates a monetizable emotion?'"*
- 성공적 퍼즐 게임은 실패가 **"내가 아쉬운"** 감정을 만듦, **"게임이 불공정한"** 감정이 아님
- Block Blast: 랜덤 블록이지만 "전략적으로 대비할 수 있었다"는 인식
- ChromaBlocks: 이 밸런스가 올바른가? → 테스트 필요

## 4. ChromaBlocks에 없는 것들

### 🔴 Critical — 있어야 경쟁 가능

#### C1. 데일리 스트릭 시스템
- **있는 곳**: Woodoku, Wordle, Duolingo, 거의 모든 성공 캐주얼 게임
- **효과**: D7 리텐션 40-60% 향상 (업계 벤치마크)
- **구현**: 매일 첫 게임 완료 시 🔥 증가. 7일/30일/100일 마일스톤 보상
- **심리**: 손실 회피(깨면 아깝다) + 매몰 비용(이미 30일 했는데) + 자기 정체성("나는 매일 하는 사람")

#### C2. 게임오버 시 "아쉬움" 극대화
- **있는 곳**: Block Blast (점수 비교 화면), Woodoku (스트릭 경고)
- **현재 ChromaBlocks**: 게임오버 화면이 감정을 충분히 자극하는가?
- **필요한 것**:
  - 최고기록과 현재 점수의 **시각적 비교** (진행 바, 얼마나 가까웠는지)
  - "X점만 더 했으면 최고기록!" 메시지
  - 즉시 재시작 버튼이 가장 크고 눈에 띄게
  - 이전 판의 "빅 플레이" 하이라이트 리플레이 (선택)

#### C3. "빅 플레이" 순간의 임팩트 강화
- **있는 곳**: Block Blast (멀티라인 연쇄), Woodoku (줄+박스 동시), Pixel Flow (트레이 관리)
- **Deconstructor of Fun**: *"What kind of puzzle failure creates a monetizable emotion?"* → 반대로, 어떤 성공이 "중독적 쾌감"을 만드는가?
- **필요한 것**:
  - 2줄+ 동시 클리어 시 **카메라 줌 + 슬로모션 + 큰 파티클**
  - 콤보 체인 시 **화면 전체가 반응하는** 이펙트 (배경색 변화, 화면 흔들림)
  - "AMAZING!" "INCREDIBLE!" 같은 **감탄사 텍스트** 팝업
  - 사운드 에스컬레이션: 콤보가 쌓일수록 음이 올라감 (도-레-미-파-솔)

### 🟡 High Priority — 경쟁 우위를 만드는 것

#### H1. Adventure/스토리 모드
- **Block Blast의 킬러 피처**: *"adventure mode allows you to explore different stories"*
- Pocket Gamer: *"you'd be surprised how many developers credit adding stories or other narrative elements with helping them succeed"*
- **구현 아이디어**:
  - 퍼즐 레벨 형식: "이 보드를 주어진 블록으로 클리어하라"
  - 레벨 맵 진행 (1→2→3... 지도 위 이동)
  - 간단한 캐릭터/배경 스토리 (블록 월드를 구하는 여정?)
  - 스타 시스템 (★★★) → 수집 욕구

#### H2. 소셜 비교 시스템
- **있는 곳**: Block Blast (Reddit 커뮤니티 자발적 점수 공유), Wordle (결과 공유)
- **필요한 것**:
  - 주간 리더보드 (같은 시드로 경쟁)
  - "결과 카드" 자동 생성 → SNS 공유 (Wordle 방식)
  - 친구 대결 (같은 시드 1:1)
- **심리**: 관계성(Relatedness) 충족 + 바이럴 확산

#### H3. 이중 클리어 목표 (보드 구조)
- **Woodoku/Blockudoku의 핵심**: 줄 클리어 + 3×3 박스 클리어
- **왜 중요**: 단일 목표(줄 클리어만) < 이중 목표(줄 + 구역)
  - 모든 수에 "선택"이 생김 → 전략적 깊이 ↑
  - 자이가르닉 효과 발동 지점 2배 → 중독성 ↑
- **ChromaBlocks 적용**:
  - 10×10 보드 유지하면서도 2×5 또는 5×5 "존" 표시 가능
  - 존 완성 시 보너스 점수 + 특별 이펙트
  - 또는 "색상 완전 제거" 보너스 (한 색상 블록 모두 클리어)

#### H4. 오프라인 가용성 + 제로 마찰
- **Block Blast**: 오프라인 플레이 가능, 광고 강제 없음
- **핵심**: 앱 아이콘 탭 → 3초 이내 게임 시작 가능해야 함
- 로딩 화면, 업데이트 팝업, 공지사항 등 = 모두 마찰
- 마찰 1초 추가 = 유저 10% 이탈 (업계 벤치마크)

### 🟢 Nice to Have — 차별화 요소

#### N1. ASMR/촉감 피드백 강화
- **Woodoku**: 나무 "톡" 소리가 차별화 핵심
- 블록 배치 사운드를 극도로 만족스럽게 → "사운드만으로 중독"
- 햅틱 패턴: 배치(짧은 탭), 클리어(중간 진동), 콤보(리드미컬 패턴)

#### N2. 동적 배경/테마 변화
- 플레이 중 점수 구간마다 배경 색상/파티클이 변화
- 1000점: 차분한 네이비 → 5000점: 별이 떠오름 → 10000점: 우주 배경
- 시각적 진행감 = "더 가보고 싶다" 동기

#### N3. "부활" 메커니즘 (수익화 연계)
- 게임오버 시 보상형 광고 1회로 보드 일부 클리어 + 계속
- **Deconstructor of Fun**: *"Match-3 works because the fail state feels recoverable. You almost won. You feel unlucky, not stupid. Spending to continue feels acceptable."*
- 블록 퍼즐에서도 동일: "이 한 줄만 비워줬으면 계속할 수 있었는데" → 부활 광고 시청

#### N4. 시즌/이벤트 시스템
- 주간 특별 룰 (이번 주는 블록이 다 L자 모양!)
- 시즌 테마 (봄: 벚꽃 파티클, 겨울: 눈 이펙트)
- 한정 보상 → FOMO(Fear of Missing Out) → 복귀 동기
